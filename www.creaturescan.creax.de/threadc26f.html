cshelper (Firefox-Extension)       

Ich habe mich mal selber daran gemacht den cshelper von GrößterNehmer auf das Redesign anzupassen.

Mit diesem Plugin werden die Daten von Firefox automatisch nach creatureScan übertragen. Neuerdings kann man auch einstellen, dass statt der Zwischenablage eine TCP-Verbidnung zur Übertragung genutzt werden soll. Damit kann man wieder ungehindert arbeiten ohne dass sich dauern der Inhalt der Zwischenablage ändert!

Beschreibung im Wiki: cshelper

cshelper 1.3f jetzt neu für Firefox 8

Hinweis: Aufgrund der häufigen Versions-Updates von Firefox ist in diesem Forums-Thread nicht immer die aktuellste cS-Helper-Version aufgeliset. Auf der Download-Seite findet ihr immer die aktuellste Version von cS-Helper die auch Zeitnah mit den FF-Updates veröffentlicht werden:
