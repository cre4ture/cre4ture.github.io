Bei einer Neuinstallation sieht es wie folgt aus:
---------------------------------------------------------------------------------

Also nach der Auswahl des Plug-Ins beendet Vista das Programm und gibt dann die Fehlermeldung aus:

creatureScan.exe funktioniert nicht mehr
Das Programm wird aufgrund eines Problems nicht richtig ausgeführt. Das Programm wird geschlossen und Sie werden benachrichtigt, wenn eine Lösung verfügbar ist.

[Programm schließen]
----------------------------------------------------------------------------------

Bei einem Start nach Einrichtung des Benutzers:
-------------------------------------------------------------------------------
Beendet Vista das Programm direkt nach dem Start wieder und gibt dann die Fehlermeldung aus:

creatureScan.exe funktioniert nicht mehr
Das Programm wird aufgrund eines Problems nicht richtig ausgeführt. Das Programm wird geschlossen und Sie werden benachrichtigt, wenn eine Lösung verfügbar ist.

[Programm schließen]
-------------------------------------------------------------------------------

Für alle die das gleiche Problem haben: Ich hab mir momentan eine Verknüpfung auf den Desktop gelegt. Dort kann man unter Eigenschaften der Verknüpfung einstellen, dass diese immer im Adminmodus ausgeführt werden soll. Und dann gehts.
